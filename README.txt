# Zeugnisnoten-Rechner (PWA)

Diese Version ist als Progressive Web App (PWA) vorbereitet. Nach dem Hochladen auf GitHub Pages kannst du sie auf Android direkt **installieren** und **offline** nutzen.

## Dateien
- `index.html` — App
- `manifest.webmanifest` — PWA-Metadaten
- `sw.js` — Service Worker fürs Offline-Caching
- `icons/` — App-Icons (192/512)

## Deployment (GitHub Pages)
1. Neues Repo auf GitHub anlegen (z. B. `zeugnisnoten-rechner`).
2. Alle Dateien aus diesem Ordner hochladen (oder die ZIP hier entpacken und per Drag & Drop ins Repo).
3. In den Repo-Einstellungen: **Settings → Pages → Branch** auf `main` und Ordner `/ (root)` stellen → **Save**.
4. Die Seite ist nach wenigen Sekunden unter der eingeblendeten URL erreichbar.

## Installation auf Android (Chrome)
1. Die GitHub-Pages-URL auf dem Smartphone öffnen.
2. In der Adressleiste oder im Menü `⋮` auf **Installieren** / **App installieren** tippen (oder „Zum Startbildschirm hinzufügen“).
3. Danach läuft die App im Vollbild und **offline**, sobald sie einmal geöffnet wurde.

Viel Erfolg!
